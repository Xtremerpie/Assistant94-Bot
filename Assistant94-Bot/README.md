# ⚡ Installation

## 1. Clone Repo

```bash
git clone YOUR_REPO_LINK
```

---

## 2. Install Requirements

```bash
pip install -r requirements.txt
```

---

## 3. Add Bot Token

Create `.env` file:

```env
TOKEN=YOUR_DISCORD_BOT_TOKEN
```

---

## 4. Run Bot

```bash
python Assistant_dcbot.py
```
